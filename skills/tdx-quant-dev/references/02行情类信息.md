# [#](https://help.tdx.com.cn/quant/docs/markdown/mindoc-1ctuhthaq5qmg/#通达信tq可以获取如下数据)通达信TQ可以获取如下数据

**股票数据**：我们拥有所有A股、港股、美股、新三板等上市公司的股票行情数据、基本信息、财务数据、市场交易数据等。
**基金数据**：我们目前提供了多种在交易所上市的基金的行情、净值等数据，包含ETF、LOF、REITs等。
**指数数据**：我们支持沪深京指数、板块指数、港股指数、中证国证指数，包括指数的行情数据以及成分股数据。具体见get_index_stocks。
**行业等板块**：我们支持按通达信行业、概念、风格板块获取成分股，具体见获取A股板块代码列表get_sector_list及获取板块成份股get_stock_list_in_sector。
**期货数据**：我们支持中金所、郑商所、大商所、上期所、上期能源、广期所的行情数据，并包含历史产品的数据。
**宏观数据**：我们提供全方位的宏观数据（.HG），为投资者决策提供有力数据支持。
**可转债和期权等数据**：我们提供完整的沪深京可转债类数据、沪深个股期权等数据。
**数据信息文件包**：我们提供各类特定的数据信息文件包。具体见download_file。

get_market_data(field_list: List[str] = [],

| 参数          | 是否必选 | 参数类型  | 参数说明                                                     |
| :------------ | :------- | :-------- | ------------------------------------------------------------ |
| field_list    | N        | List[str] | 字段筛选，传空则返回全部                                     |

- 返回dict { field1 : value1, field2 : value2, ... }
- field1, field2, ... ：数据字段
- value1, value2, ... ：pd.DataFrame 数据集，index为stock_list，columns为time_list
- 各字段对应的DataFrame维度相同、索引相同
- 只有dividend_type传入为none时，会返回有效的前复权因子ForwardFactor
- 一次最多返回24000条数据，要获取完整分钟线需要多次分批获取
