### [#](https://help.tdx.com.cn/quant/docs/markdown/TdxQuant.md/mindoc-1h10m001ic888.html#根据股票-获取指定时间段内的专业财务数据-与基础财务数据不同-需要先在客户端中下载专业财务数据)根据股票，获取指定时间段内的专业财务数据，与基础财务数据不同，需要先在客户端中下载专业财务数据

```python
get_financial_data(stock_list: List[str] = [],

					end_time: str = '',

```

| 参数        | 是否必选 | 参数类型  | 参数说明                                                     |
| :---------- | :------- | :-------- | ------------------------------------------------------------ |
| stock_list  | Y        | List[str] | 证券代码列表例如 `["600519.SH"]`                             |

| report_type | N        | bool      | 按截止日期还是公告日期筛选，可选值：`'announce_time'`（按公告日期筛选）或 `'tag_time'`（按报告期筛选） |
