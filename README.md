# Agent Skills Repository

本仓库提供了一系列标准化的 AI Agent 技能 (Skills)，旨在增强 Agent 在特定领域的开发能力。遵循 [Agent Skills](https://agentskills.io) 标准。

## 技能列表

| Skill 名称 | 描述 |
| :--- | :--- |
| `android-development` | 提供 Android Kotlin 开发的最佳实践、架构规范 (Clean Architecture, MVVM)、依赖注入 (Koin) 及测试模版。 |
| `coding-standards` | 提供通用编码规范与最佳实践，涵盖异常处理、日志规范、单元测试、代码复用 (DRY)、类型安全及构建验证等，提升代码质量与可维护性。 |

## 安装指南

使用以下命令安装所需技能：

```bash
npx skills add https://github.com/wzcxbg/develop-agent-skills --skill android-development
npx skills add https://github.com/wzcxbg/develop-agent-skills --skill coding-standards -a trae
```
